# agents/

This directory contains agent authority and lifecycle specifications.

AI agents are autonomous software entities that act within delegated authority from sovereign principals. This directory will hold:

- Agent authority pattern documentation
- Agent lifecycle models (provisioning, activation, operation, termination)
- Multi-agent orchestration sovereignty patterns
- Agent identity and accountability frameworks
- Escalation and human-in-the-loop design guidance

See [schemas/agent-authority.schema.json](../schemas/agent-authority.schema.json) for the base authority schema.
See [docs/agent-sovereignty.md](../docs/agent-sovereignty.md) for the doctrine.
