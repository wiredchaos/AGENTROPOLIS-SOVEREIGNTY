# memory/

This directory contains memory sovereignty principles, structures, and guidance.

AI memory — conversation history, behavioral patterns, preferences, and contextual state — is among the most sensitive data in AI-native systems. This directory will hold:

- Memory categorization frameworks
- Memory lifecycle documentation (creation, access, retention, deletion)
- Memory portability specifications
- Guidance on memory consent patterns

See [schemas/memory.schema.json](../schemas/memory.schema.json) for the base memory schema.
See [docs/individual-sovereignty.md](../docs/individual-sovereignty.md) for memory ownership principles.
