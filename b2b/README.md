# b2b/

This directory contains business-to-business sovereignty patterns and documentation.

B2B sovereignty governs automated data flows and system integrations between organizations. This directory will hold:

- B2B interoperability contract patterns
- B2B authority chain documentation
- Mutual accountability frameworks
- Revocation and offboarding process guides
- B2B audit and compliance patterns

See [schemas/interoperability.schema.json](../schemas/interoperability.schema.json) for the base contract schema.
See [docs/b2b-b2ai-sovereignty.md](../docs/b2b-b2ai-sovereignty.md) for the doctrine.
