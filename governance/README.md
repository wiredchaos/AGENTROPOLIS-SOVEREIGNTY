# governance/

This directory contains governance frameworks and check structures for AI sovereignty.

Governance is the accountability and oversight layer of the sovereignty stack. This directory will hold:

- Governance framework documentation
- Governance check pattern libraries
- Escalation process documentation
- Accountability chain audit guides
- Multi-stakeholder governance models
- Sovereignty compliance checklists

See [schemas/governance-check.schema.json](../schemas/governance-check.schema.json) for the base governance check schema.
See [docs/sovereignty-doctrine.md](../docs/sovereignty-doctrine.md) Section 4 for the sovereignty stack context.
