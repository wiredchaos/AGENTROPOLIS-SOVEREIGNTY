# wallets/

This directory contains value and permission wallet concept documentation.

In AI sovereignty, a "wallet" is a sovereign control structure — not just a financial instrument. Wallets may hold permissions, verifiable credentials, capability grants, data asset references, and value attribution records.

This directory will hold:

- Wallet concept documentation
- Permission wallet patterns
- Capability wallet patterns
- Data asset wallet frameworks
- Value attribution and tracking models

See [schemas/wallet-control.schema.json](../schemas/wallet-control.schema.json) for the base wallet control schema.

**Note:** This directory does not and will not contain real wallet implementations, private keys, live financial logic, or production transaction flows.
