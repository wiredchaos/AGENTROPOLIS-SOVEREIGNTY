# b2ai/

This directory contains business-to-AI sovereignty patterns and documentation.

B2AI sovereignty governs how organizations interact with AI systems controlled by third parties — AI platforms, APIs, and services. This directory will hold:

- B2AI interoperability contract patterns
- AI provider governance requirements
- Data residency and retention frameworks for AI-processed data
- Model training opt-out documentation
- AI-to-AI (A2A) interaction governance
- Exit and data portability guidance for AI platform relationships

See [schemas/interoperability.schema.json](../schemas/interoperability.schema.json) for the base contract schema.
See [examples/sample-b2ai-flow.json](../examples/sample-b2ai-flow.json) for an illustrated example.
See [docs/b2b-b2ai-sovereignty.md](../docs/b2b-b2ai-sovereignty.md) for the doctrine.
