# identity/

This directory contains identity model specifications for sovereign entities.

Identity is the foundation of the sovereignty stack. This directory will hold:

- Identity model documentation
- DID method profiles for different entity types
- Identity attribute disclosure frameworks
- Credential schema references

See [schemas/identity.schema.json](../schemas/identity.schema.json) for the base identity schema.
